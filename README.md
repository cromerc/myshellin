# myshellin
myshellin es un proyecto semestral del ramo Sistemas Operativos. El propósito de este proyecto es crear un shell básico.

## Autores
- Christopher Cromer
- Raúl Raúl Hernandez

## Requisitos software
- glibc
- make 
- clang

## Requisitos informe
- latex
- pdflatex

## Compilar

### Software
make myshellin

### Informe
make informe
